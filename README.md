Single-Cell RNA-seq Analysis of GSE212447
Overview

This repository contains the analysis workflow for the single-cell RNA sequencing (scRNA-seq) dataset GSE212447 from the NCBI Gene Expression Omnibus (GEO). The goal of this project is to explore cellular heterogeneity within the dataset, identify transcriptional differences between conditions, and characterize cell populations using clustering and reference-based annotation.

The analysis was performed primarily in R using the Seurat framework, along with additional Bioconductor tools for downstream analysis and cell type annotation.

Dataset source:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE212447
